# HTML Notes

## What is Web?
 - The World Wide Web (WWW or "the web") is a system of interlinked hypertext documents (webpages) and media accessed via the internet using browsers like Chrome or Firefox.

## What is an Internet?
- The Internet is a global network of connected devices that uses standard protocols to enable communication, information sharing, and digital services.
## What is WebDevlopment?
- Web devlopment is process of building and maintaing website and web applications.

## What is a Website?
- A website is a collection of interconnected, publicly accessible web pages—including text, images, and videos—that share a single domain name and are hosted on a web server.
## What is Web Application?
- A web application (web app) is interactive software application software, such as Gmail, Trello, or online banking, that runs on a remote server and is accessed entirely through a web browser without requiring local installation

## What is HTML?
- Structure of a webpage
- Uses tags like <h1>, <p>, <div>

## What is Tag ?
- The tag tells the browser which type of content in present and how to display it . Using angele (< >) brackets.

## Key Concepts
- Semantic tags: header, footer, main, section
- Forms: input, label, button
- Accessibility: alt text, aria labels
