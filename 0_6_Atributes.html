Atributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

For example, in the tag <input type="text" id="name" name="name">, the attributes are type, id, and name. The type attribute specifies the type of input field, the id attribute provides a unique identifier for the element, and the name attribute specifies the name of the input field, which is used when submitting form data.

Some common attributes include:

  • class: Used to specify one or more class names for an element, which can be used for styling with CSS or for targeting with JavaScript.
  • id: Used to specify a unique identifier for an element, which can be used for styling with CSS or for targeting with JavaScript.
  • href: Used in anchor tags to specify the URL of the page the link goes to.
  • src: Used in image tags to specify the URL of the image to be displayed.
  • alt: Used in image tags to provide alternative text if the image cannot be displayed.
  • title: Used to provide additional information about an element, often displayed as a tooltip when the mouse hovers over the element.
  • style: Used to apply inline CSS styles to an element.
